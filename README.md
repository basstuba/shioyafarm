# custom-laravel
## .env変更部分
 ```
   APP_NAME=プロジェクト名

   APP_TIMEZONE=Asia/Tokyo

   APP_LOCALE=ja

   APP_FALLBACK_LOCALE=ja

   APP_FAKER_LOCALE=ja_JP

   DB_CONNECTION=mysql

   DB_HOST=mysql

   DB_PORT=3306

   DB_DATABASE=任意のデータベース名

   DB_USERNAME=任意のユーザー名

   DB_PASSWORD=任意のパスワード
   ```
> [!Warning]
> compose.yamlのMYSQL_DATABASE,MYSQL_USERNAME,MYSQL_PASSWORDと合わせること
